ezglib.gl

array-gl-type
(array-gl-type value)
Returns the equivalent WebGl constant contained in a typed array.

aspect
(aspect gl)
Gets the current aspect ratio of the gl context.

bind-texture!
(bind-texture! gl texture)(bind-texture! gl texture tex-unit)
Binds the texture to the context to be used by an ezglib shader.

blend-func!
(blend-func! gl src dest)
Sets the gl context to use the given blend function.

buffer
(buffer gl & {:keys [data target usage item-size], :as args})
Creates a gl buffer with initialized data.

:data must be a typed array-buffer.

:target (optional) must be either ezglib.render.gl/array-buffer (default) or ezglib.render.gl/element-array-buffer.

:usage (optional) must be either ezglib.render.gl/static-draw (default) or ezglib.render.gl/dynamic-draw.

:item-size (optional).
clear!
(clear! gl)
Clears the gl context.

clear-color!
(clear-color! gl r g b a)(clear-color! gl)
Clears the color buffer of the gl context.

clear-depth!
(clear-depth! gl depth)(clear-depth! gl)
Clears the depth buffer of the gl context.

clear-stencil!
(clear-stencil! gl index)(clear-stencil! gl)
Clears the stencil buffer of the gl context.

context-height
(context-height gl)
Gets the height of the gl context's drawing buffer.

context-lost?
(context-lost? gl)
Returns whether the context was lost.

context-width
(context-width gl)
Gets the width of the gl context's drawing buffer.

create-context
(create-context canvas)
Creates the opengl context.

current-shader
(current-shader gl)
Gets the ezglib shader currently bound to the gl context.

draw!
(draw! gl & {:keys [uniforms attributes textures shader draw-mode first count blending? blend-src blend-dest capabilities element-buffer element-offset], :as opts})
Draws to the gl context.

draw-arrays!
(draw-arrays! gl draw-mode first count)(draw-arrays! gl draw-mode count)
Draws arrays to gl context.

draw-elements!
(draw-elements! gl buffer draw-mode count offset)
Draws the elements of bound attribute arrays to gl context.

enabled?
(enabled? gl capability)
Checks if a certain capability is enabled.

free-shader
(free-shader gl program)
Frees a shader from memory.

free-texture
(free-texture gl tex)
Frees a loaded texture

get-canvas
(get-canvas gl)
Returns the canvas used by the gl context.

get-viewport
(get-viewport gl)
Returns the current viewport for a given gl context as a map with the form:

{:x,
 :y,
 :width,
 :height}
load-shader
(load-shader gl frag vert)
Loads an ezglib shader from fragment and vertex shader source.

load-texture
(load-texture gl image min-filter mag-filter mipmap?)
Loads a texture.

native-aspect
(native-aspect gl)
Gets the native aspect ratio of the gl context, i.e. the aspect ratio of the canvas.

reset-viewport!
(reset-viewport! gl)
Resets the viewport to be the same size as the canvas.

set-attribute!
(set-attribute! gl location {:keys [buffer normalized? stride type offset components-per-vertex], :as opts})
Sets an attribute on the current ezglib shader.

set-attribute*!
(set-attribute*! gl location buffer)
Quickly sets an attribute on the current ezglib shader with default parameters.

set-capabilities!
(set-capabilities! gl & capabilities)
Enables or disables capabilities.

set-capability!
(set-capability! gl capability enabled?)
Enables or disables a capability.

set-uniform!
(set-uniform! gl location value)
Sets a uniform on the current ezglib shader.

value should be a typed array or a type that extends ezglib.protocol.ITypedArray.

location must be the keyword form of the uniform name.
set-viewport!
(set-viewport! gl {:keys [x y width height], :as viewport})
Sets the current viewport for a given gl context. Expects a map with the form:

{:x,
 :y,
 :width,
 :height}

Returns the width, in texels, of the texture.
